Emergency Broadcast System

EBS plays when a channel has no scheduled media, when playback fails, or when a weather capture error occurs. Pick off-air video and audio independently — for example, color bars with a beep tone, or TV static with white noise.

Self-hosted WeatherStar (Docker)

Host WeatherStar locally. Do not point production channels at a public demo URL.

FinTV can start official Docker images for WeatherStar 4000+ and 3000+. Docker must be installed and the Jellyfin user must be able to run docker. If Jellyfin runs in Docker, mount /var/run/docker.sock.

WeatherStar 4000+ (ws4kp)

netbymatt/ws4kp · port 8080

WeatherStar 3000+ (ws3kp)

netbymatt/ws3kp · port 8083

WeatherStar base URL

FinTV appends each weather channel's latitude and longitude as ?lat=…&lon=…. Use Start + Use URL above for local Docker, or enter a LAN URL manually.

For testing only, you may use a public hosted demo (not for production):

  • https://weather.jmthornton.net/
  • https://4000.weatherstar.dev/
  • https://weatherstar3000.netbymatt.com/

Weather channels use headless Chromium via Playwright and EBS background music. Linux: Playwright uses mcr.microsoft.com/playwright:v1.49.0-jammy. Localhost WeatherStar URLs are rewritten to host.docker.internal for container access.

Examples: http://127.0.0.1:8080 (ws4kp), http://127.0.0.1:8083 (ws3kp). Do not include lat or lon here.
